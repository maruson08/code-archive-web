# 📚 코드 아카이브 웹앱

이 웹앱은 수업 시간에 작성한 알고리즘/자료구조 실습 코드를 정리해 보여주는 아카이빙 페이지입니다.

## 포함된 내용
- 거스름돈 문제
- Prim 알고리즘
- 최대 부분배열 합(분할정복/DP)
- 8차시 DP 발표 코드
- Sudoku 백트래킹 풀이
- 문자열 내 최소 반복 단위
- Boyer-Moore / KMP / Rabin-Karp
- 5주차 문제 5번/6번
- 해시 테이블: 선형 탐사, 제곱 탐사, Double Hashing, Rehashing, 3-B

## 사용 방법
1. 브라우저에서 index.html 열기
2. 검색창으로 원하는 주제 검색
3. 카드 클릭 시 코드 상세 보기
4. 복사 또는 다운로드 가능

## 파일 구성
- index.html: 페이지 구조
- styles.css: 스타일
- script.js: 검색/필터/모달 기능
- data.js: 아카이빙 데이터
