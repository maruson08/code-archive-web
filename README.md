# 📚 코드 아카이브 (Code Archive)

## 개요

이 프로젝트는 대학 수강 과정에서 작성한 알고리즘 및 자료구조 코드를 아카이빙하고 관리하는 웹 애플리케이션입니다.

## 기능

- 🔍 **검색 기능**: 코드 제목, 주제, 설명으로 검색 가능
- 🎯 **카테고리 필터**: 동적계획법, 분할정복, 해시 테이블 등 카테고리별 필터링
- 📋 **코드 보기**: 모달 창에서 코드 전문 확인
- 📋 **코드 복사**: 한 번의 클릭으로 코드를 클립보드에 복사
- ⬇️ **다운로드**: 코드를 Python 파일로 다운로드
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두에서 최적화

## 코드 카테고리

- **동적계획법 (DP)**: LCS, 거스름돈 문제, 최대 부분배열 합 등
- **분할정복**: 최대 부분배열 합, 가장 가까운 두 점의 거리 등
- **해시 테이블**: 선형 탐사, 체이닝, 나눗셈 해싱 등
- **문자열 탐색**: Boyer-Moore, KMP, Rabin-Karp
- **백트래킹**: N-Queens 문제
- **자료구조**: Heap Tree, 우선순위 큐
- **그래프**: 다익스트라 알고리즘
- **그리디 알고리즘**: 거스름돈 문제

## 수강 과정

- 6주차: 알고리즘 성능 분석
- 7주차: 분할정복
- 8차시: 동적계획법 (DP)
- 9차시: 백트래킹
- 정보 3주차: 문자열 처리 (모둠별 프로젝트)
- 정보과학 5주차: 알고리즘 최적화 (모둠별 프로젝트)
- 자료구조: Queue, 해시테이블

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Data**: JSON (JavaScript 배열)
- **Design**: CSS Grid, Flexbox, CSS Variables

## 사용 방법

1. `index.html`을 웹 브라우저로 열기
2. 상단 검색창에서 코드 검색
3. 카테고리 드롭다운에서 필터링
4. 원하는 코드 카드 클릭하여 상세 내용 확인
5. "코드 복사" 또는 "다운로드" 버튼으로 활용

## 파일 구조

```
code-archive-web/
├── index.html       # 메인 HTML 파일
├── styles.css       # 스타일시트
├── script.js        # JavaScript 기능
├── data.js          # 코드 데이터

```

## 주요 기능 상세

### 검색
- 실시간 검색으로 즉시 결과 업데이트
- 제목, 주제, 설명에서 모두 검색 가능

### 필터링
- 카테고리별 필터링으로 특정 주제의 코드만 보기
- 검색과 필터링을 함께 사용 가능

### 모달 뷰
- 클린한 인터페이스로 코드 전체 내용 확인
- 관련 정보(주제, 카테고리, 원본 파일) 함께 표시

### 코드 관리
- 한 번의 클릭으로 코드 복사
- 자동으로 Python 파일명 생성하여 다운로드
